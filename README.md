### 1.7 分析问题

**Q1：Adam 和 SGD 的收敛速度有何差异？**

Adam优化器相比SGD具有更快的收敛速度，利用自适应学习率机制，能更快找到最优解。

**Q2：学习率对训练稳定性有什么影响？**

学习率过大（如0.01）会导致训练不稳定，损失震荡；学习率过小会导致收敛缓慢。

**Q3：Batch Size 对模型泛化能力有什么影响？**

较大的Batch Size提供更稳定的梯度估计，较小的Batch Size引入更多噪声，可能有助于跳出局部最优。

**Q4：Early Stopping 是否有效防止了过拟合？**

Early Stopping有效防止了过拟合，当验证损失连续5个epoch没有改善时自动停止训练。

**Q5：数据增强是否提升了模型的泛化能力？**

数据增强显著提升了模型的泛化能力，通过随机旋转、平移等增强，模型学习到了更多变化模式。
